## 更新日志格式说明

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/) 语义化版本规范。

### 版本号格式

```
主版本号.次版本号.修订号

示例: 2.0.0
- 主版本号: 不兼容的重大更改
- 次版本号: 向后兼容的新功能
- 修订号: 向后兼容的问题修复
```

### 版本状态标签

| 标签 | 说明 |
|------|------|
| `**breaking**` | 破坏性变更 |
| `**feature**` | 新功能 |
| `**fix**` | 问题修复 |
| `**perf**` | 性能改进 |
| `**refactor**` | 重构 |
| `**docs**` | 文档更新 |
| `**build**` | 构建相关 |
| `**ci**` | CI/CD 相关 |
| `**chore**` | 其他更改 |
